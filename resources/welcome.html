Tcl 9.1.0

This is the Tcl 9.1.0 installer for macOS.

Click "Customize" on the next panel to select which components will be installed.

NOTE: This installer will create or overwrite files in the directories Library/Frameworks or /usr/local/bin. If you already have an installation of Tcl/Tk 9.1 it will be modified.

For more information about Tcl and Tk, please visit:
https://www.tcl-lang.org/about/
